# Slidev Structure（建议结构）

```text
slides/
  deck.md
  public/
    figures/          # 软链接或复制最终图（来源于 figures/**/out）
  style/
    theme.css
```

## deck.md 结构建议

- Title
- Problem & Gap
- Method intuition (schematic)
- Results (multi-slide, 1 conclusion each)
- Limitations
- Takeaways
- Backup slides（备用）
