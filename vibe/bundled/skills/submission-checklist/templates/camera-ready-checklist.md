# Camera-ready / Proof Checklist（相机就绪 / 校样）

## A. 版式与一致性
- [ ] 所有图表编号与引用一致（Fig. 2 / Figure 2）
- [ ] 所有缩写首次出现已定义
- [ ] 单位/符号/变量记号全篇一致
- [ ] 参考文献完整、无缺失字段（作者/年份/期刊/DOI）

## B. 图表最终规格
- [ ] 矢量图无栅格化（PDF/EPS/SVG）
- [ ] TIFF 满足 300–600dpi（按期刊要求）
- [ ] 颜色在灰度打印下仍可区分（必要时用不同线型/点型）

## C. 构建与归档
- [ ] 可复现构建命令记录（`make`/`latexmk`/`quarto render`）
- [ ] 最终提交 zip 可在干净环境下解压并编译（LaTeX）
- [ ] 归档：提交版本 tag、构建产物、投稿系统回执
